# dotfiles
ターミナルを使いやすくするdotfilesの設定たち

# セットアップ方法

## リポジトリのclone
ホームディレクトリにcloneする。
ホームディレクトリ以外にcloneするとうまく動きません。

```
$ git clone git@github.com:ytebine07/dotfiles.git
```

## セットアップシェルの実行

    ./setup.sh

## Macに必要なディレクトリ作成

最近のMacには `/usr/local/bin` が無いようなので作成する

```
$ sudo mkdir /usr/local/bin
```
